# Java Programming Algorithms

This repository contains Java implementations of basic programming concepts and logic-building exercises.  
The programs are written to practice core programming fundamentals such as arrays, loops, case statements, bitwise operations, and searching techniques.

## Topics Covered

### 1. Arrays
Programs related to array operations including:
- Input and output of array elements
- Sum of array elements
- Finding maximum element
- Multi-dimensional arrays
- Counting even and odd numbers
- Addition of two matrices

### 2. Bitwise Operations
Examples demonstrating:
- Bitwise shift operators
- Arithmetic operations with bit manipulation

### 3. Case Statements
Programs using `switch-case` for:
- Simple number mapping
- Basic calculator operations

### 4. Loops
Examples of different looping constructs:
- While loop
- Do-while loop
- For loop

Programs implemented using loops:
- Multiplication table
- Sum of natural numbers
- Factorial
- Armstrong number
- Palindrome number
- Prime number
- Fibonacci series
- Reverse of a number

### 5. Searching
Programs related to searching digits in a number:
- Check if a digit exists in a number
- Count frequency of a digit in a number

## Technologies Used

- Java
- Basic programming concepts
- Console-based programs

## Purpose of This Repository

This repository is created for:
- Practicing Java programming fundamentals
- Understanding algorithmic thinking
- Building a strong programming foundation
